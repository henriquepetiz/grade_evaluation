Este é um programa simples em Python desenvolvido para converter notas numéricas (0-10) em conceitos de letras (A-F), seguindo a lógica de escalas acadêmicas internacionais.

# Funcionalidades

- **Validação de Entrada:** O programa garante que apenas notas entre 0 e 10 sejam aceitas.
- **Suporte a Decimais:** Aceita notas com pontos decimais (ex: 8.5).
- **Classificação Instantânea:** Converte a nota para o sistema de letras:
  - **A**: 9.0 - 10.0
  - **B**: 7.0 - 8.99
  - **C**: 5.0 - 6.99
  - **D**: 3.0 - 4.99
  - **E**: 0.6 - 2.59
  - **F**: Abaixo de 0.59
